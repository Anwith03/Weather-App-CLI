# 🌦️ Weather App (CLI)

A simple Command Line Interface (CLI) Weather Application developed using **Python**. This application fetches real-time weather information for any city using the **WeatherAPI** service.

---

## 📌 Features

- 🌍 Search weather by city name
- 🌡️ Current temperature
- 🤗 Feels like temperature
- 💧 Humidity
- 💨 Wind speed
- ☁️ Weather condition
- ❌ Handles invalid city names
- ⚡ Fast API response
- 💻 Easy-to-use command line interface

---

## 🛠️ Technologies Used

- Python 3
- Requests Library
- WeatherAPI
- VS Code

## 📦 Installation

Clone the repository

```bash
git clone https://github.com/YOUR_USERNAME/Weather-App-CLI.git
```

Go to the project folder

```bash
cd Weather-App-CLI
```

Install required packages

```bash
pip install -r requirements.txt
```

---

## ▶️ Run the Project

```bash
python weather.py
```
