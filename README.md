# Weather App CLI

A simple Command Line Interface (CLI) Weather Application built with Python.

## Features

- Get real-time weather information
- Search weather by city name
- Displays:
  - City
  - Region
  - Country
  - Temperature
  - Feels Like
  - Humidity
  - Wind Speed
  - Weather Condition

## Technologies Used

- Python
- Requests
- python-dotenv
- WeatherAPI

## Installation

```bash
pip install -r requirements.txt
```

## Create a .env file

```
API_KEY=YOUR_API_KEY
```

## Run

```bash
python weather.py
```
